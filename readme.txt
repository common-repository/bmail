Bmail is a mass email client that allows you to send emails through your site.

== Description ==
Bmail allows you to send individual emails to multiple recipients. You can send an email to one person or separate emails by commas to send an email to each recipient's email address individually.
Supports HTML, and embedded media from the wordpress editor. Bmail lets you send single emails or mass-mails to multiple recipients without everyone appearing in the To: field.

Bmail now supports hotkeys for more personalized e-mail messages to multiple recipients.

Bmail is available in:

* English


== Installation ==

The plugin is simple to install:

1. Download `Bmail.zip`
2. Unzip
3. Upload `Bmail` directory to your `/wp-content/plugins` directory
4. Go to the plugin management page and enable the plugin
5. Go to Settings > Bmail to start Bmailing
